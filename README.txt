Documentation available @ https://github.com/Shashwatraj3951/Cryptr

This tool is tested on :--

1. Kali Linux
2. Parrot
3. Ubuntu
4. Windows

This tool helps you to encrypt your personal data with password. 

Installtion:-

1. apt get update
2. apt get install git
3. git clone https://github.com/Shashwatraj3951/Cryptr.git
4. cd Cryptr
5. bash cryptr.bash

For encrypting data :-

1. bash cryptr.bash encrypt <file with its path>
2. Then enter a security password.

For decrypting data :-

1. bash cryptr.bash decrypt <file.aes with its path>
